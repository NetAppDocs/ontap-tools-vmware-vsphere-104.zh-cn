= ONTAP tools for VMware vSphere概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以通过 ONTAP 工具的维护控制台管理您的应用程序、系统和网络配置。您可以更改您的管理员密码和维护密码。您还可以生成支持包、设置不同的日志级别、查看和管理 TLS 配置以及启动远程诊断。

部署ONTAP tools for VMware vSphere后，您应该安装 VMware 工具以访问维护控制台。你应该使用 `maint`作为您在部署期间配置的用户名和密码，以登录ONTAP工具的维护控制台。您应该使用 *nano* 在维护或 root 登录控制台中编辑文件。


NOTE: 您应该为 `diag`用户同时启用远程诊断。

您应该使用已部署的ONTAP tools for VMware vSphere *Summary* 选项卡来访问维护控制台。当您选择image:../media/launch-maintenance-console.gif["播放按钮"]，维护控制台启动。
