= 更新 vCenter Server 凭据
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用维护控制台更新 vCenter Server 实例凭据。

.开始之前
您需要拥有维护用户登录凭证。

*关于此任务*

如果您在部署后更改了 vCenter Server 的凭据，则需要使用此过程更新凭据。

.步骤
. 从 vCenter Server 打开 VASA Provider 控制台。
. 以维护用户身份登录。
. 进入 `2`选择系统配置菜单。
. 进入 `8` 更改 vCenter 凭据。
