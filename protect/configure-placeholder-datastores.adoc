= 配置占位数据存储
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您应该配置一个占位数据存储，以便在恢复站点的 vCenter 清单中为受保护的虚拟机 (VM) 预留一个位置。占位数据存储不需要很大，因为占位虚拟机很小，仅占用几百 KB 甚至更少的空间。

.开始之前
* 您应该已经连接了受保护的站点和恢复站点。
* 您应该已经配置了资源映射。


.步骤
. 登录到 vCenter Server 并选择 *Site Recovery* > *Sites*。
. 选择受保护的站点并选择*管理*。
. 在管理选项卡中选择 *占位符数据存储* > *新建* 来创建一个新的占位符数据存储。
. 选择适当的数据存储并选择*确定*。
+

NOTE: 占位数据存储可以是本地的或远程的，并且不应被复制。

. 重复步骤 3 至 5 为恢复站点配置占位数据存储。
