= 配置文件夹映射
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您应该映射受保护站点和恢复站点上的文件夹，以实现它们之间的通信。

.开始之前
您应该已经连接了受保护的站点和恢复站点。

.步骤
. 登录到 vCenter Server 并选择 *Site Recovery* > *Sites*。
. 选择受保护的站点并选择*管理*。
. 在管理选项卡中选择*文件夹映射* > *文件夹*图标来创建新的文件夹映射。
. 在创建文件夹映射向导中，执行以下操作：
+
.. 选择*自动准备具有匹配名称的文件夹映射*并选择*下一步*。
.. 选择受保护和恢复站点所需的数据中心对象，然后选择*添加映射*。
.. 映射创建成功后选择*下一步*。
.. 选择之前用于创建反向映射的对象，然后选择*完成*。




.结果
文件夹映射页面显示受保护站点资源和恢复站点资源。您可以对您环境中的其他网络执行相同的步骤。
