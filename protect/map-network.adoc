= 配置网络映射
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您应该在两个站点上配置资源映射，例如 VM 网络、ESXi 主机和文件夹，以便将受保护站点中的每个资源映射到恢复站点上的相应资源。

您应该完成以下资源配置：

* 网络映射
* 文件夹映射
* 资源映射
* 占位数据存储


.开始之前
您应该已经连接了受保护的站点和恢复站点。

.步骤
. 登录到 vCenter Server 并选择 *Site Recovery* > *Sites*。
. 选择受保护的站点并选择*管理*。
. 在管理选项卡中选择*网络映射*>*新建*来创建新的网络映射。
. 在创建网络映射向导中，执行以下操作：
+
.. 选择*自动准备具有匹配名称的网络映射*并选择*下一步*。
.. 选择受保护和恢复站点所需的数据中心对象，然后选择*添加映射*。
.. 映射创建成功后选择*下一步*。
.. 选择之前用于创建反向映射的对象，然后选择*完成*。




.结果
网络映射页面显示受保护站点资源和恢复站点资源。您可以对您环境中的其他网络执行相同的步骤。
